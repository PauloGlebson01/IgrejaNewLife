⛪ Igreja Batista New Life — Website Oficial

Site institucional desenvolvido para apresentar a Igreja Batista New Life, seus valores, horários de cultos, liderança e localização.

📌 Sobre o Projeto
Este projeto é um website responsivo criado com foco em:

Apresentação institucional da igreja
Divulgação de cultos e eventos
Exibição da liderança pastoral
Integração com redes sociais
Facilidade de acesso à localização

🚀 Tecnologias Utilizadas
HTML5
CSS3
JavaScript (Vanilla JS)
Font Awesome (ícones)
Google Fonts

📂 Estrutura do Projeto
📁 projeto
 ├── index.html      # Estrutura principal do site
 ├── style.css       # Estilização e responsividade
 ├── script.js       # Interações e funcionalidades
 └── assets/         # Imagens e ícones

 ✨ Funcionalidades
📱 Menu mobile responsivo
🔗 Navegação com scroll suave
📍 Integração com Google Maps
📲 Links diretos para redes sociais
📊 Exibição de estatísticas sociais
🧭 Botões de navegação intuitivos

🖥️ Seções do Site
Home — Apresentação principal
Sobre — Missão, visão e propósito
Cultos — Horários e informações
Pastores — Liderança da igreja
Localização — Endereço com mapa integrado
